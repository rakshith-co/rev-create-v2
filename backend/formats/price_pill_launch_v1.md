# Price-Pill Launch

## Blueprint

Full-bleed twilight facade or aerial hero, graded to one deep tone; project lockup top-left, one emotional headline mid-left ('The corner of the city that stays quiet.'). Below it, a single quiet rounded pill carries the only number on the canvas — '3 BHK from ₹1.62Cr*' — with an optional second micro-pill for possession year. A compliant dark footer strip runs full-width at the base in micro type: RERA registration number, project/promoter name, disclaimer and site URL — the strip is a fixed system component so every export ships compliant by default. No starbursts, no strike-throughs, one seal maximum.
